==== Guide d'Installation PFM_Blockchain ====

Ce répertoire contient des ressources pour l'installation et l'utilisation de l'application PFM_Blockchain:

1. PFM_Blockchain_Guide.md - Guide complet au format Markdown
2. PFM_Blockchain_Guide.docx.md - Version du guide optimisée pour la conversion en DOCX

Pour convertir le guide en DOCX:
- Ouvrez PFM_Blockchain_Guide.docx.md dans Microsoft Word
- Enregistrez le document au format DOCX

Ou utilisez Pandoc (si installé):
```
pandoc PFM_Blockchain_Guide.docx.md -o PFM_Blockchain_Guide.docx
```

Le guide inclut:
- Instructions d'installation avec ganache-cli
- Configuration de MetaMask pour se connecter à la blockchain locale
- Description des fonctionnalités de chaque exercice
